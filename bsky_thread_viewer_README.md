# Bluesky Thread Viewer

This bookmarklet opens a Bluesky thread viewer for the current URL, displaying the full branching tree of any Bluesky thread in a new window.

## Installation

### Easy Mode
1. Go to the [Bluesky Thread Viewer Bookmarklet Installer](https://austegard.com/web-utilities/bookmarklet-installer.html?bookmarklet=bsky_thread_viewer.js)
2. Drag the link to your bookmarks bar
### Hard Mode
1. Copy the entire JavaScript code from the bookmarklet file at https://github.com/oaustegard/bookmarklets/blob/main/bsky_thread_viewer.js
2. Go to [Bookmarklet Installer](https://austegard.com/web-utilities/bookmarklet-installer.html)
3. Paste the code
4. Name the bookmarklet
5. Drag the link to your bookmarks bar

## Usage

When viewing a Bluesky post or thread, click the bookmarklet. It will open a new tab containing the full thread viewer with branching structure, making it easier to read complex conversations, such as those with hundreds of replies.
